# Thessis Ecommerce

## Installing

### Step 1
Run the venv
```
pip install pipenv
pipenv shell
```

### Step 2
Install th requirments
```
pip install -r requirements.txt
```

### Step 3
Create SQLite databse, run migrations
```
cd store
python manage.py migrate

```
### Step 4
Run Django dev server
```
python manage.py runserver

```
